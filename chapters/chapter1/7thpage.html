JavaScript Accessors (Getters and Setters)


Getters

Syntax

let obj = { get propName() { // getter, the code executed on getting obj.propName },

Example

let user = { name: "John", surname: "Smith", get fullName() { return `${this.name} ${this.surname}`; } }; alert(user.fullName); // John Smith

Setter

Syntax

let obj = { set propName(value) { // setter, the code executed on setting obj.propName = value } };

Example

let user = { get name() { return this._name; }, set name(value) { if (value.length < 4) { alert("Name is too short, need at least 4 characters"); return; } this._name = value; } }; user.name = "ABCD"; alert(user.name); // ABCD user.name = ""; // Name is too short...

Why Using Getters and Setters?

1.It gives simpler syntax

2.It allows equal syntax for properties and methods

3.It can secure better data quality

4.It is useful for doing things behind-the-scenes
